# Stocks_Market_Dashboard
This repository contains a Power BI dashboard built for analyzing stock market performance across companies such as AAPL, GOOG, MSFT, and NFLX. The dashboard provides insights into closing prices, daily returns, volatility patterns, and overall performance within a selected date range.

# Features
Close price trends across selected tickers
KPIs for Latest Close, Performance %, and 30-Day Volatility
Daily returns visualization to monitor short-term fluctuations
Volatility (30D) trends over multiple months
Interactive filters for stock tickers and custom date ranges
